# Tendencias de Netflix

## Objetivo:
Analizar la evolución, distribución y características del catálogo de series y películas de Netflix para identificar tendencias de contenido,
preferencias regionales y patrones deproducción en la plataforma. 

## Dataset:
https://www.kaggle.com/datasets/shivamb/netflix-shows

## Tecnologías empleadas:
* Jupyter Notebook
* Pandas
* Matplotlib
* Seaborn
* Numpy
* PowerBI
